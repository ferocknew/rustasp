# ASP 运行时现代内建对象设计（Rust 实现）

在 Classic ASP 的基础上，可以在 Rust ASP Runtime 中新增现代对象，提供更丰富的功能，同时保持跨平台和安全。

---

## 一、设计原则

1. **全局可用或通过 CreateObject 创建**

    * Json、Math 等可作为全局对象直接访问
    * 可选通过 Server.CreateObject 创建

2. **跨平台安全**

    * 避免 Windows-only COM / DLL / ActiveX
    * 文件访问受沙箱限制

3. **模块化**

    * 每个对象独立文件，不超过 500 行
    * 易于扩展和维护

4. **统一接口**

    * 所有对象实现 `AspObject` trait
    * 统一支持方法调用和属性访问

---

## 二、可新增对象列表

| 对象/功能            | 说明            | Rust 模块               | 调用方式                                                       |
| ---------------- | ------------- | --------------------- | ---------------------------------------------------------- |
| Json             | 对象/数组序列化和解析   | `objects/json.rs`     | Json.Parse / Json.Stringify / Server.CreateObject("Json")  |
| HttpClient       | 请求外部 HTTP API | `objects/http.rs`     | HttpClient.Get / Post / Server.CreateObject("HttpClient")  |
| Math             | 数学函数集合        | `objects/math.rs`     | Math.Sin / Math.Cos / Math.Sqrt / Math.Random              |
| Timer / DateTime | 时间和日期处理       | `objects/datetime.rs` | Timer.Now / Timer.AddSeconds / 全局对象                        |
| Crypto           | 哈希与加密         | `objects/crypto.rs`   | Crypto.SHA256 / Crypto.MD5 / Server.CreateObject("Crypto") |
| Logger           | 日志记录          | `objects/logger.rs`   | Logger.Info / Logger.Warn / Logger.Error / 全局对象            |

## 三、示例：JSON 对象设计

```rust
pub struct JsonObject;

impl AspObject for JsonObject {
    fn call_method(&mut self, name: &str, args: Vec<Value>) -> Result<Value, RuntimeError> {
        match name.to_lowercase().as_str() {
            "parse" => Ok(Value::Object(parse_json(&args[0])?)),
            "stringify" => Ok(Value::String(stringify_json(&args[0])?)),
            _ => Err(RuntimeError::MethodNotFound),
        }
    }

    fn get_property(&self, _name: &str) -> Option<Value> {
        None
    }
}
```

> 同理，其他对象可以封装相应 Rust 库实现具体功能。

## 四、白名单注册示例

```rust
registry.register("Json", || {
    Value::Object(Rc::new(RefCell::new(JsonObject::new())))
});

registry.register("HttpClient", || {
    Value::Object(Rc::new(RefCell::new(HttpClient::new())))
});
```

---

## 五、实现建议

1. **先实现全局必需对象**（Json、Math、DateTime、Logger）
2. **可选通过 Server.CreateObject 创建**（HttpClient、Crypto 等）
3. **模块化拆分，每个对象独立文件**
4. **统一 AspObject trait 接口**，支持方法调用、属性访问
5. **保证沙箱和安全性**

---

## 六、总结

通过在 Rust ASP Runtime 中新增现代内建对象：

* 拓展 Classic ASP 功能，支持 JSON、HTTP、加密、数学、日志等
* 保持跨平台和安全沙箱
* 支持全局对象和白名单扩展对象模式
* 易于维护和未来扩展
