## 优先级建议

### P0 - 核心功能（已实现）
- 类型转换函数
- 数学函数
- 基本字符串函数
- 日期时间函数
- 数组函数

### P1 - 重要功能
- 字节处理函数（LenB, LeftB, RightB, MidB, InStrB, AscB, ChrB）
- 脚本引擎信息函数
- Escape/Unescape

### P2 - 可选功能
- Erase
- 改进 DateDiff（需要完整的日期解析）
- 改进 DateValue/TimeValue（需要完整的日期解析）

### P3 - 不建议实现
- InputBox, MsgBox（客户端UI）
- CreateObject, GetObject（COM依赖）
- GetRef, Execute, ExecuteGlobal（安全风险）
- LoadPicture（客户端）
