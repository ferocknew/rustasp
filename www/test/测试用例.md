# ASP VBScript 运行时测试用例

- **项目**: asp-lite (Rust 实现的 Classic ASP 运行时)
- **服务器地址**: http://127.0.0.1:8080
- **测试时间**: 2026-03-05, 更新于 2026-03-07

## 三、控制流测试

### 3.1 If/Then/Else
**文件**: `test_005_if.asp`

```asp
<%
Dim score
score = 85

If score >= 90 Then
    Response.Write "优秀"
ElseIf score >= 80 Then
    Response.Write "良好"
ElseIf score >= 60 Then
    Response.Write "及格"
Else
    Response.Write "不及格"
End If
%>
```

**预期结果**: 显示 "良好"

---

### 3.2 Select Case
**文件**: `test_006_select.asp`

```asp
<%
Dim dayOfWeek
dayOfWeek = 3  ' 假设 3 = 星期三

Select Case dayOfWeek
    Case 1
        Response.Write "星期一"
    Case 2
        Response.Write "星期二"
    Case 3
        Response.Write "星期三"
    Case Else
        Response.Write "其他"
End Select
%>
```

**预期结果**: 显示 "星期三"

---

### 3.3 For 循环
**文件**: `test_007_for.asp`

```asp
<%
Dim i
For i = 1 To 5
    Response.Write "第 " & i & " 次循环<br>"
Next
%>
```

**预期结果**: 输出 5 行，显示循环次数

---

### 3.4 While 循环
**文件**: `test_008_while.asp`

```asp
<%
Dim count
count = 1

While count <= 3
    Response.Write "Count = " & count & "<br>"
    count = count + 1
Wend
%>
```

**预期结果**: 输出 3 行

## 七、数组测试

### 7.1 静态数组
**文件**: `test_015_array_static.asp`

```asp
<%
Dim fruits(2)
fruits(0) = "苹果"
fruits(1) = "香蕉"
fruits(2) = "橙子"

Dim i
For i = 0 To 2
    Response.Write fruits(i) & "<br>"
Next
%>
```

**预期结果**: 列出三种水果

---

### 7.2 动态数组
**文件**: `test_016_array_dynamic.asp`

```asp
<%
Dim numbers()
ReDim numbers(2)
numbers(0) = 10
numbers(1) = 20
numbers(2) = 30

ReDim Preserve numbers(4)
numbers(3) = 40
numbers(4) = 50

For Each num In numbers
    Response.Write num & "<br>"
Next
%>
```

**预期结果**: 输出 1 到 5 的数字

### 9.2 Form 提交
**文件**: `test_020_form.asp`

```asp
<%
If Request.ServerVariables("REQUEST_METHOD") = "POST" Then
    Dim username
    username = Request.Form("username")
    Response.Write "提交的用户名: " & username
Else
%>
<form method="post" action="">
    <input type="text" name="username">
    <input type="submit" value="提交">
</form>
<% End If %>
```

**预期结果**: POST 后显示提交的用户名
